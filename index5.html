Bootstrap

Es una herramienta que proporciona interactividad en la página, por lo que ofrece una serie de componentes que facilitan la comunicación con el usuario.

Es un framework CSS utilizado en aplicaciones front-end — es decir, en la pantalla de interfaz con el usuario— para desarrollar aplicaciones que se adaptan a cualquier dispositivo.

¿Como imprementarlo?

Se puede visitar la página oficial de Bootstrap y descargar el paquete. De esta forma, puedes tener los archivos de este framework en tu servidor.
En los archivos de instalación de Bootstrap puedes ver un directorio donde están las hojas de estilo (css), otro directorio para los ficheros JavasScript (js) y un directorio de fuentes (fonts) donde se encuentra la fuente por defecto para los iconos que utiliza Bootstrap (Glyphicons).

Para poder trabajar con Bootstrap tan solo debes añadir en la cabecera, dentro de la etiqueta head o antes de la etiqueta de cierre body de tu archivo HTML las hojas de estilo y el archivo JavaScript de Bootstrap.

¿Como funciona?

Existe un archivo principal llamado bootstrap.css, que tiene una definición para los estilos utilizados. Básicamente, la estructura del framework se compone de dos directorios:
css: contiene los archivos necesarios para la estilización de los elementos y una alternativa al tema original.

js: contiene la parte posterior del archivo bootstrap.js (original y minificado), responsable de la ejecución de aplicaciones de estilo que requieren manipulación interactiva.​ Para asignarle una característica a un elemento, simplemente debemos declarar la hoja de estilo bootstrap junto con su js, y si se quiere modificar algún estilo en específico se hace por medio de las clases.

Ventajas

1. Es una herramienta muy sencilla de utilizar e intuitivo, adaptada a cualquier programador y que permite crear grandes sitios web en poco tiempo.

2. Provee de todas las reglas CSS para que nuestro sitio web se adapte de manera dinámica a la mayoría de las pantallas (es responsive).

3. Cuenta con múltiples implementaciones externas: WordPress, Drupal, SASS o jQuery UI.

4. Hace uso de componentes y servicios que han sido creados por la comunidad web. Como, por ejemplo: HTML5 shim, Normalize.css, OOCSS, jQuery UI, LESS o GitHub.

5. Bootstrap nos ofrece distintas herramientas para extender nuestro framework, así como adaptarlo a nuestras necesidades y características, es decir que nosotros lo podemos editar.

6. Ofrece ”themes” o temas de estilo por defecto los cuales se pueden mantener o modificar.

7. Bootstrap utiliza LESS, una ampliación a las hojas de estilo CSS, que funciona igual que un lenguaje de programación.

Skeleton

Skeleton es un framework ligero para css creado por Dave Gama el 15 de mayo del 2011.
Este Framework está centrado para las personas que están pensando en diseñar un proyecto web pequeño o cuando solo se necesitan utilizar las funciones más básicas, gracias a esto,este framework tiende a ser ligero en cuanto a peso y los recursos que utiliza para su implementación, Por defecto Skeletonutiliza elementos base de HTML además de incluir GRID.

La tipografía y fuentes se ajusta con REMS, haciendo que esta sea responsiva.

Utilidades.

Skeleton viene con estilos predefinidos que son adicionales. Algunos son los siguientes:

  • button clase: Le permite diseñar una etiqueta “a”(Para referenciar hyperlinks) elemento ancla “()” como botón
  • background-color: Permite darle al botón un color azul usando la clase “button-primary”.
  • u-pull-left: Permite agregar un elemento flotante hacia la izquierda.
  • u-pull-right: Permite agregar un elemento flotante hacia la derecha.
  • u-cf: Es una clase ayudante que permite borrar algunos flotantes.

Ventajas.

  • Es Ligero y fácil de personalizar en el mercado.
  • Es rápido y sencillo de usar.
  • Es multiplataforma y fácil de desarrollar.

Desventajas

  • Tiene una limitación por su código indefinido.
  • Tiene un menor control visual.
  • Tiene una importación innecesaria de códigos.

Foundation

Este framework está dividido en módulos, lo cual significa que en el momento de su descarga, el usuario puede elegir que módulos necesita para surespectivo trabajo.
Es un marco responsivo que contiene muchos elementos HTML y CSS ademásde extensiones opcionales de JavaScript.Se pueden utilizar plantillas y fragmentos de código para diseñar interfaces deusuario.

Ventajas.

  • Mejora la velocidad de ejecución.
  • Es mas rápido para codificar.
  • Foundation goza de una reputación comoplataforma de desarrollo muy liviana.Incluso, debido a la estructura modular yamencionada y a la libertad en la selección,es posible reducir el tamaño de esteentorno de trabajo aún más.

Desventajas.

  • Trabajar con estas hojas de estilo noresulta fácil para quienes no tenganexperiencia, que demanda graninversión de tiempo y trabajo.
  • Tiene una muy reducida selecciónde plantillas y falta de soporte paralas versiones mas antiguas.
  • La integración de componentesexternos.

Pre-mini

Es el conjunto de reglas de Normalize.css v7.0.0 para tratar las diferencias entre navegadores, al tiempo que aplica algunos valores predeterminados de buen gusto, como el uso de la pila de fuentes nativas para averiguar la mejor fuente para cada dispositivo.

mini.css incluye muchas funciones en un paquete pequeño, mientras que se basa únicamente en CSS, lo que significa no demos preocuparnos por ningún conflicto con otras bibliotecas de JavaScript que pueda estar usando.

Ventajas

  • No es demasiado pesado (46kb).
  • Documentación clara y búsqueda sencilla.
  • Sistema de grid (columnas) clónico de Bootstrap.
  • Se aplica la gran mayoría de los estilos a las etiquetas. 3 temas incluidos (Original, Dark y Nord).
  • No necesitas JavaScript para nada.

Desventajas

  • Es un formato simple. Todo lo básico viene incluido, pero el resto tendrás que fabricarlo.
  • Las listas podrían estar más elaboradas.
  • No cumple el formato típico para los menús de headers (nav, ul, li, a).

Pure Css

En el año 2013, Yahoo lanza Pure CSS.

Una estructura básica para el desarrollo de interfaces web que es considerada como una alternativa a Bootstrap de gran calidad, pero que también se puede utilizar en conjunto con este framework.

El framework de Yahoo asigna a cada módulo (independientemente de si se trata del diseño regular o de reglas específicas) un nombre de clase estándar con el prefijo “pure” y los interpreta como submódulos.

¿Qué es Pure.CSS?

Es un framework CSS para facilitar el front-end de los desarrolladores web.

La base de este framework está en Normalize.css, un proyecto donde se pretende que todos los navegadores muestren exactamente lo mismo (bueno, o casi exactamente).

Caracteristicas.

  • Tiene una cuadrícula (grid) que se adapta a todas las resoluciones.
  • Botones con estilo, tanto con a como con button.
  • Permite construir menús horizontales, verticales, de varios niveles.
  • Facilidad para crear formularios, con estilos comunes.
  • Varios formatos comunes de tablas.
  • Look minimalista, y fácil de modificar.
  • Permite modificar con skin builder.
  • Ocupa poco tamaño.

Pure se puede colocar en el HTML de la siguiente manera:

link rel="stylesheet" href="http://yui.yahooapis.com/pure/0.4.2/pure-min.css"

El Sitio web de Pure está construido con su framework, por lo que su código minimalista y bien desarrollado lleva a una guía sencilla de navegar.
